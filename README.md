# 🎮 Ludex

Ludex es una app hecha con React Native (Expo) para buscar videojuegos, ver precios en distintas tiendas y guardarlos en una wishlist.

La idea es tener en un solo sitio:
- 🔎 Búsqueda de juegos
- 💸 Comparación de precios
- ⭐ Wishlist persistente
- 🔔 Avisos de precio

---

## 🚀 Tecnologías

- React Native (Expo)
- TypeScript
- Zustand (estado global)
- AsyncStorage (persistencia)
- RAWG API (datos de juegos)
- CheapShark API (precios)

---

## 📱 Funcionalidades

- Buscar juegos por nombre  
- Ver detalles (imagen, géneros, rating, descripción)  
- Comparar precios entre tiendas  
- Añadir juegos a wishlist  
- Guardar alertas de precio  
- Comparador entre juegos
